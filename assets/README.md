# Image assets

The homepage references the following images from the Claude Design project
(“4Ma Construction”, file `4Ma Construction.dc.html`):

- `4ma-logo.png` — company logo (header)
- `1b-hero.png` — hero photo
- `1b-g1.png` — featured project (large)
- `4ma-real-render.png` — project photo
- `1b-g3.png` — project photo

Export these from the design project's `assets/` folder (or drop in your own
photos with the same filenames). Until they exist, the site shows styled
placeholder blocks instead of broken images.
